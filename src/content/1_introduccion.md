## 🎨 1. ¿Qué es CSS?

**CSS** significa **Cascading Style Sheets** (en español: *Hojas de Estilo en Cascada*).

Sirve para definir:

- colores
- tamaños
- márgenes
- tipografías
- layouts
- animaciones

👉 **En resumen:** CSS controla cómo se ve una página web.

## 🧱 2. Sintaxis básica de CSS

CSS se compone de **reglas**.

```css
selector {
  propiedad: valor;
}
```

**Ejemplo:**

```css
h1 {
  color: blue;
}
```

Esto significa: todos los `<h1>` serán azules.

### 🔎 Partes de una regla CSS

```css
h1 {
  color: blue;
}
```

| Parte      | Qué hace   |
| ---------- | ---------- |
| `h1`       | Selector   |
| `color`    | Propiedad  |
| `blue`     | Valor      |
