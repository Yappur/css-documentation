# Guía 1: Fundamentos de CSS

## ¿Qué es CSS?
CSS (Cascading Style Sheets) se usa para dar estilo a páginas web.

## Cómo usar CSS
- Inline
- Internal (en <style>)
- External (archivo .css)

## Selectores básicos
- Etiqueta: p { }
- Clase: .clase { }
- ID: #id { }

## Propiedades básicas
- color
- background
- font-size
- margin
- padding
- border

## Modelo de caja
- content
- padding
- border
- margin
